Zeige deinen Teamspeakserver als Widget in deiner Wordpress-Sidebar an.

== Description ==

Fügt ein Sidebarwidget hinzu, welches Guido van Biemen's (aka MrGuide@NL) Teamspeak Display Preview Release 3 (GPL) anzeigt. Nutzt AjaxCore 1.2.2 from http://ajaxcore.sourceforge.net zur Aktualisierung

Das Widget wird unter [mein Blog](http://www.gehirnfurz.com/tsviewerwidget/) (zur Refernz).

== Installation ==
1. Stellt sicher, dass Ihr [widgets plugin](http://automattic.com/code/widgets/) installiert habt, wenn Ihr die Wordpressversion 2.0/2.1 einsetzt.
2. Lade das Pluginpaket herunter.
3. Entpacke es im Verzeichnis `/wp-content/plugins/`
4. Aktiviere das Plugin im Bereich 'Plugins' deines Wordpressmenüs
5. Gehe zu Themes/WIdgets und ziehe das Plugin in deine Sidebar
6. Konfiguriere das Widget ... falls du einen local Teamspeakserver anbinden willst, der mit Standardeinstellungen läuft, dann brauchst du nichts konfigurieren.
7. Hab Spass.
